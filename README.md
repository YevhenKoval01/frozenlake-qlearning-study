# FrozenLake Q-Learning Study

A team machine learning experiment focused on comparing reinforcement learning settings for the FrozenLake environment. The repository contains merged experiment results and the final report.

## Tech Stack

- Python
- Reinforcement learning
- Q-learning
- CSV-based result analysis
- PDF reporting


## Project Structure

```text
data/
  wyniki_zespolu_scalone.csv
reports/
  raport_koncowy.pdf
README.md
.gitignore
```

## How To Use

1. Open `data/wyniki_zespolu_scalone.csv` in Excel, Google Sheets, Python, or another analysis tool.
2. Review the experiment parameters, hypotheses, training results, evaluation results, and conclusions.
3. Open `reports/raport_koncowy.pdf` for the final written summary.
